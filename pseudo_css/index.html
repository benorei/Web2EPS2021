Today, we will mess with some Pseudo Classes.

Let's first dive deeper into CSS.

If we think of CSS in terms of selectors, properties, and values; a typical CSS entry would look like this: 

            
                selector {
                    property: value;
                }

To style the above, we need to add two tags together. We will call this CSS Combinators.

What are Pseudo Classes?

Again, if we think of CSS in terms of selectors, properties, and values; a pseudo class is an add-on to a selector. 

            
                selector:pseudoclass {
                    property: value;
                }

We use them for links

If we want our link to look different, we can simply add it in our CSS as follows: 

            
                a {
                    color: red;
                }

But if we want each of our link states to look different, then we will use a pseudo class like:

  • :visited
  • :hover
  • :active

We can still override each of these properties with classes. 

                
                    a.special {
                        font-weight: 800;
                        text-decoration: none;
                        color: purple;
                    }
                    
                    a:hover.special {
                        color: blueviolet;
                    }

Now our special links look like this: I'm special.

We can use Pseudo Classes on any tag.

We can use IDs or Classes to assign pseudo-styles to anything, like: 

            
                #hover-div {
                    color: white;
                    background: darkslategray;
                    padding: 10px;
                    text-align: center;
                }
                
                #hover-div:hover {
                    background: black;
                }

Now we have a hoverable div.

This div has a hover, what do you think?

We can even call on specific selectors depending on their hierarchy.

To make only the first p red, we can use: 

            
                #div-with-children p:first-child {
                    color: red;
                }

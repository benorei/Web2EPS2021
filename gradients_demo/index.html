CSS Gradients Demo

This demo will show how to make gradients in CSS. Gradients are a useful tool that can be used to add a bit of spice to your color design and palette. There are many types of gradients and this guide will show how to make multiple. This demo's final project is a box with a button that will have different gradients on its background, its face, and its hover.

Setup

In a new HTML page, make a button inside a div, and give it a little CSS to center it.

# Index.html <div class="button-container"> <button>Button</button> </div> # #Style.css .button-container { text-align: center; }

Linear Gradient

Now that we've got that set up, let's think about some gradients. First, we're going to make the background div a nice linear gradient that goes diagonally.

You can choose any colors you want, but I'm going to go with a nice light to dark blue.

The syntax for a linear gradient is: background: linear-gradient(color1, color2)

#Style.css .button-container { background: linear-gradient(lightblue, blue); } #Let's also make the button container look a little nicer too. .button-container { padding: 20px; border-radius: 10px; }

Okay, that did something! It looks kind of boring going from top to bottom though, so let's rotate it. We can rotate it by adding angle in front of the colors like so:

.button-container { background: linear-gradient(45deg, lightblue, blue); }

Look at that! It's rotated 45 degrees. You can also rotate it in any direction by changing the number of degrees. To make the rotation based on the shape of the box, you can replace 45deg with to (top | bottom) right | left. You can also add more colors by just adding more colors!

.button-container { background: linear-gradient(to left, lightblue, blue, lightgreen, green); }

Pick whatever colors and orientation you think fit best, and let's move on. I've picked green, light blue, and 60deg.

Radial Gradient

Radial gradients have one color in the center that fades to another color in the outside. The syntax is quite similar to the linear gradient. Let's use this to make our button face. (We'll also add a little extra base CSS to give us more room to see our gradient.

button { background: radial-gradient(red, rebeccapurple); padding: 20px; border-radius: 10px; border: none; width: 300px; }

Well that's certianly something! With radial gradients it's especially important to pick colors that look good together.

I do think that ovaly shaped thing looks weird. We can change the shape of the gradient by adding the shape before the colors, like so:

button { background: radial-gradient(circle, #E58500, #002642); }

We can also change the position of our "circle":

button { background: radial-gradient(circle at top left, #E58500, #002642); color: white; }

Conic Gradient

People might not know it's a button though - so let's add a different gradient as the hover effect. We're going to use a conic gradient, which works similarly to the radial one.

button:hover { background: conic-gradient(lime, aqua, blue, magenta); }

Looks interesting. You may not like the "cutoff", but this can be fixed by putting the first color a second time at the end of the list.

button:hover { background: conic-gradient(lime, aqua, blue, magenta, lime); }

Conclusion

But with a few color palletes (I recommend coolors.co) and your new knowledge of CSS gradients, you will be able to make some awesome looking stuff. (And remember! These gradients can be applied to anything, not just buttons! Play around with linear-, radial-, and conic-gradient() and see what you can apply them to!)

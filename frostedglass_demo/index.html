A new, effortless design.
Based on the demo by Logan

I ended up making a few changes but this demo is super cool. Super simple yet creates a really sleek, modern looking effect.

    Changes I made:
  • Used percentages instead of rem and vh for width and height
  • Removed grid display in favor of just normal html
  • Margin defined by tag, not by the frosted glass class
  • Additional "fit," "minpad," and "mx" classes to fit width to content, shrink padding, and define top margin respectively.
